# KIBO Dropship Sales Kit

Internal sales enablement page. Embedded in Google Sites for the Marketing & Sales teams.

## Files
- `index.html` — fully self-contained sales kit (all fonts, icons, and logos inlined)
- `.nojekyll` — tells GitHub Pages to skip Jekyll processing

## Deploy
1. Upload both files to the repo root (keep the leading dot on `.nojekyll`)
2. Settings → Pages → Source: `Deploy from a branch` → `main` / `/ (root)`
3. Wait ~60 seconds for the green deploy banner
4. Embed the resulting URL in Google Sites via Insert → Embed → By URL
